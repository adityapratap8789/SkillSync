# Coding Techniques and Styles

## Frontend
- **React**: Utilized for building dynamic and responsive user interfaces, allowing for component-based architecture.
- **MUI (Material-UI)**: Provides a set of React components that implement Google's Material Design, ensuring a consistent and modern look.
- **React Router**: Manages navigation and routing within the application, enabling single-page application behavior.

## Backend
- **Node.js**: Offers a scalable and efficient environment for server-side operations, leveraging JavaScript for both client and server.
- **Express.js**: A minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications.

## Styling
- **CSS-in-JS**: Used for component-level styling with MUI's `sx` prop, allowing for dynamic styling based on props and state.
- **Responsive Design**: Ensures the application is accessible and usable on a variety of devices, from mobile phones to desktop computers.

## File Upload
- **Drag-and-Drop**: Implemented to provide an intuitive and user-friendly way to upload files.
- **Validation**: Ensures that only PDF files are uploaded and checks for file size constraints to prevent issues with large files.

These techniques and styles were chosen to create a modern, efficient, and user-friendly application, leveraging the strengths of React and Node.js ecosystems.
